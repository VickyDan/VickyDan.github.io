给2017上半年定几个小目标

在今天这个2017已经过了六分之一，刚开学不到一周的时候给自己的2017上半学期定几个小目标：

  1. 作为学生学习当然是第一要务（虽然之前有两三个学期过得很随意），因此要求自己从这个学期开始认真听课，不在课堂上被手机控制支配
  2. 通过计算机等级考试三级网络技术
  3. 认真学习中国大学mooc上的浙大陈越教授开设的数据结构课程以及北大唐大仕老师开设的java程序设计课程，并争取拿到90+的成绩
  4. 通过jsp+SpringMVC+MyBatis实现个人博客网站的搭建，实现博客所需的功能
  5. 坚持每天写一篇博文
    失败是成功之母，坚持不懈才是让我们从失败走向成功的桥梁。一步步养成好习惯，让自己成为更优秀的人。
