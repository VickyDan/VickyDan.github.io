java相关的基础知识

婧雪蔷薇杂记集
2017-02-28

关于java与C++的不同：
无直接指针操作；自动内存管理；数据类型长度固定；不用头文件；不包含结构和联合；不支持宏；不用多重继承；无类外全局变量；无GOTO
java运行机制；虚拟机；代码安全性检测，垃圾收集机制
javaSE应用分为Application和Applet两种，Application程序需要在jvm虚拟机上运行；Applet程序要继承JApplet，有paint()方法表示绘制没有main方法，必须嵌入到html文件中

关于java的工具：
1、javac 编译
2、java 运行
3、javaw 运行图形界面程序
4、appletViewer 运行applet程序
5、jar 打包工具
（1）编译 javac A.java
（2）打包 jar cvfm A.jar A.man A.calss(c表示创建，v表示显示详情，f表示指定文件名，m表示生成清单文件，如A.man,可任意命名常见的是MANIFEST.MF)
（3）运行 java -jar A.jar
6、javadoc 生成文档
javadoc -d 目录名 xxx.java
7、javap查看类信息(javap 类名)及反汇编（java -c 类名）

关于输入输出
1、文本界面：输入使用Scanner类，输出使用out(输入也可以使用in:System.in.read(),早期的JDK采用)
2、图形界面输入输出：
使用文本框对象（TextField）
