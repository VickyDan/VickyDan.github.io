每天几道编程题总结(1)

婧雪蔷薇杂记集
2017-08-14
-学习 -编程

题1：汇编语言中有一种移位指令叫做循环左移（ROL），现在有个简单的任务，就是用字符串模拟这个指令的运算结果。对于一个给定的字符序列S，请你把其循环左移K位后的序列输出。例如，字符序列S=”abcXYZdef”,要求输出循环左移3位后的结果，即“XYZdefabc”。是不是很简单？OK，搞定它！（来源：牛客网《剑指offer》）
解法一：该解法是做的时候首先想到的，每次左移一次都进行一个循环，时间复杂度为O(N*N)
public String LeftRotateString(String str, int n) {
if (str.isEmpty())
return “”;
char[] cstr = str.toCharArray();
for (int i = 0; i < n % cstr.length; i++) {
char temp = cstr[0];
for (int j = 0; j < cstr.length - 1; j++) {
cstr[j] = cstr[j + 1];
}
cstr[cstr.length - 1] = temp;
}
String s = new String(cstr);
return s;
}
解法二：”abcdefg”扩展为”abcdefgabcdefg”,截取n到n+len的部分即为所求字符串
public String LeftRotateString(String str, int n) {
int len = str.length();
if (len == 0)
return “”;
n = n % len;
str += str;
return str.substring(n, len + n);
}
解法三：分割法，将待左移的字符串”abcdefg”分割成两部分，需要移到末尾的为一部分，剩余的为另一部分，如，n=2,第一部分为”ab”,第二部分为”cdefg”,将第二部分加上第一部分就得到循环左移2位后的结果”cdefgab”
public String LeftRotateString(String str,int n) {
int len=str.length();
if(len==0)
return “”;
n=n%len;
String n1=str.substring(0,n);
String n2=str.substring(n,len);
return n2+n1;
}
